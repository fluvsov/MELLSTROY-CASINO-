# MELLSTROY-CASINO
# КАЗИНО МЕЛЛСТРОЙНОСТИ by fluvsov

Консольный симулятор слот-машины на Python. 

**Дата создания:** 20.06.2026  

## Особенности
* Вывод через ANSI-коды
* Три символа: чекушка, жирность, бурмалда

## Как запустить
Запустите интерактивный режим через терминал:
```bash
python -i MELLSTROY-CASINO.py
```
После запуска напишите в консоли `казино()` для прокрутки барабанов.
